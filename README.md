# Event Countdown App

## What I Built & Why

I built a simple **Event Countdown Dashboard** to help people track important dates and deadlines.

- **Choice of Design**: I wanted something that felt clean and modern, similar to the built-in timers you'd see in Windows 11. It's designed to be used in dark mode so the focus stays on the actual countdown numbers.
- **Simplifying the View**: I chose a digital-only display for the countdowns because it’s much faster to read at a glance than a traditional progress ring.
- **Color Indicators**: I used a simple red, yellow, and green color system to show which events are coming up soonest.

## What I would improve with More Time

- **Notifications**: Adding a browser sound or desktop alert when a timer ends.
- **Better Organization**: Letting users group events into folders or categories.
- **Mobile Polish**: Adding extra touch-friendly animations for mobile users.
- **Settings**: Letting users toggle between different time formats.

## Challenges Faced

1. **styling**: Getting the dark mode theme right across all components while keeping the code simple and readable.
2. **Merging Date & Time**: Writing the logic that combines separate date and time selections into a single countdown from "now."
3. **Responsive Grid**: Making sure the dashboard grid shifts cleanly between mobile and desktop views.

## Time Spent

I spent roughly **8 hours** total on the project, from the initial sketching and design work to the final code.
